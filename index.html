使用者條款

  1. 服務範圍

    本服務提供預約管理、服務統計、客戶管理及本地推播通知功能。

    用戶須自行提供真實、完整且有效的資訊，以利服務運作。

  2. 帳號與密碼

    用戶需妥善保管帳號與密碼，對帳號下所有操作負責。

    若帳號遭未授權使用，用戶應立即通知我們。

  3. 資料使用

    用戶提供的資料僅用於服務運作、功能改善及合法用途。

    包括帳號資訊、預約紀錄、服務偏好、照片或影像檔及操作紀錄。

    本服務會採取安全措施保護敏感個資，照片由 Cloudflare R2 提供伺服器端加密保護。

  4. 禁止行為

    用戶不得：

    • 上傳、散布違法、侵權或不當內容
    • 企圖破壞服務運作或竊取其他用戶資料
    • 用於非本服務授權之用途

    違反者，我們有權暫停或終止其帳號使用權。

  5. 責任限制

    本服務會盡力維持系統運作，但不保證服務絕對不中斷或無錯誤。

    對因使用服務所產生之直接或間接損失，我們在法律允許範圍內不承擔責任。

  6. 條款修改

    使用者條款如有更新，我們會在 App 內或網站通知用戶，用戶須重新同意最新版本方可繼續使用服務。

隱私權政策

隱私權政策

  1. 資料收集

    我們在提供服務時，會收集以下用戶資料：

    • 基本帳號資訊（姓名、Email、電話等）
    • 預約紀錄與服務偏好
    • 使用行為與操作紀錄（用於改善服務）
    • 照片或影像檔（用於服務紀錄與預約管理）
    • 使用行為分析資料（透過 Firebase Analytics 收集）
    • 顧客管理資料（由會員自行輸入，如顧客姓名、聯絡方式、備註等）

    這些資料僅用於提供預約管理、帳務統計、個人化服務，以及改善使用者體驗。

  2. 資料加密與保護

    為保障用戶資料安全，我們採取以下措施：

    • 敏感個人資料（帳號密碼、Email、電話等）在伺服器端進行 AES-256-CBC 加密儲存
    • 照片與影像檔上傳至 Cloudflare R2，由 R2 提供伺服器端加密保護
    • 本地推播使用必要資訊通知用戶，推播資料不包含敏感個資
    • Firebase Analytics 收集的使用行為資料不含敏感個資
    • 系統僅授權程式或管理人員可透過服務端程式存取資料

  3. 用戶權益

    用戶可隨時查詢、更新或刪除自己或客戶的個人資料，包括顧客資料與照片。

    用戶可選擇是否提供特定資訊，但部分資料為服務運作必要。

  4. 資料保留與刪除

    系統僅保留用戶資料於服務期間及法規要求期限。

    用戶要求刪除資料時，我們將於合理時間內從資料庫與備份中清除。刪除後的資料將無法復原，請用戶自行確認。

  5. 條款更新

    隱私權政策如有更新，我們會在 App 內通知用戶，用戶須再次確認同意最新版本方可繼續使用服務。
